# 프로젝트 기반 기술 학습

> 독립 프리셋
> 익숙한 프로젝트의 코드를 통해 새로운 기술/언어를 학습
>
> **플레이스홀더**:
> - `{학습 대상}` → 배우고 싶은 기술/언어 (예: Nix, Rust, Go)
> - `{익숙한 기술}` → 이미 잘 아는 기술/언어 (예: JS/TS, Python)

```
이 프로젝트에서 사용하는 {학습 대상}의 기본 이해와, 반드시 알면 좋은 핵심 개념들에 대해 알고 싶어. 이러한 지식들을 실제 내 프로젝트의 코드와 연관지으면서 나의 이해를 돕도록 해. 참고로 나는 {익숙한 기술}에 매우 익숙하니, 예시를 {익숙한 기술}에 빗대어 설명하면 훨씬 이해하기 쉬울 것 같아.
```
