# PR 피드백 반영

> 독립 프리셋
> CodeRabbit/인간 리뷰어 코멘트 triage 및 반영

```text
PR 리뷰 코멘트를 triage해.
각 항목을 유효성/타당성/복잡성/실현가능성/YAGNI/NGMI/회귀 관점으로 평가하고,
유효한 항목만 커밋해서 push해.

반영하지 않은 항목은 "기각 근거(코드/실행/요구사항)"를 답글로 남겨.
단, "사용자가 그렇게 요청했다"만으로는 보안/안정성 지적을 기각하지 마.

마지막에 이번 라운드 로그를 남겨:
- prompt version
- 반영한 코멘트 수 / 기각한 코멘트 수
- 다음 라운드로 이월한 항목
```
